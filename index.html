Notion Context Skill for AI Edge Gallery

This GitHub Pages site hosts a custom AI Edge Gallery skill that lets Gemma read unfinished tasks from Notion, sort them by due date and priority, and summarize what to tackle next.

Open SKILL.md Read Setup Notes
Use the GitHub Pages root URL in AI Edge Gallery. The right URL pattern is https://YOUR-NAME.github.io/REPO-NAME/.

Current defaults

  • Status filter: Status != Done
  • Due date column: Date
  • Priority column: Priority
  • Private database ID: supplied through the app secret or the prompt

Example prompt

What tasks are due soon, and what are 3 quick wins I can knock out?
